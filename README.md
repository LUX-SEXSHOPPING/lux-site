# LUX Site Oficial

Site institucional da **LUX Sexshopping**, desenvolvido em HTML, CSS e JavaScript.

## ✨ Recursos

- Banner em tela cheia
- Layout moderno e responsivo
- Galeria de imagens
- Botão de contato pelo WhatsApp
- Pronto para publicação no GitHub Pages

## 📁 Estrutura do projeto

```
LUXsiteOficial/
│── index.html
│── style.css
│── script.js
│── README.md
│
└── assets/
    ├── banner.jpg
    ├── foto1.jpg
    ├── foto2.jpg
    └── foto3.jpg
```

## 🚀 Publicação

1. Envie todos os arquivos para o repositório.
2. Acesse **Settings > Pages**.
3. Selecione:
   - Branch: `main`
   - Pasta: `/ (root)`
4. Salve e aguarde alguns minutos.
